개요

권봉석 부회장님께서 AI Studio에서 직접 제작하신 Travel Tailor 프로토타입을 기반으로, Voice Memo 피드백을 반영하여 프로덕션 수준의 앱으로 고도화하였습니다.

기존 프로토타입의 핵심 컨셉은 유지하면서, 사용자 경험과 실용성을 대폭 강화했습니다. 캘린더 기반 날짜 선택, 실제 예약 사이트 연동, 카테고리 기반 일정 추천 등 피드백에서 언급하신 주요 개선 사항을 반영하였습니다.

Voice Memo 피드백 요약

부회장님의 Voice Memo 피드백을 분석하여 핵심 개선 방향을 도출하였습니다.

Voice Memo 원문 (발췌)

"맨 처음에는 목적지, 출발지 이런 거 말고 여행 기간을 하면 그때부터 내가 1일차에 해야 될 To Do List를 쫙 해서 그 이동, 숙소 이런 거 다 걸러내고, 1일차에는 그때부터 휴식, 운동, 관광 이런 몇 가지 카테고리로 제안을 해주면 내가 거기서 어떤 카테고리를 정하면 그때부터 더 상세한 부자재로 진화를 해서..."

1
여행 기간을 가장 먼저 설정하도록 플로우 변경
2
일차별 To Do List 형태로 준비 항목 생성
3
이동/숙소 등 카테고리별 필터링 구현
4
카테고리 선택 후 상세 옵션으로 진화(심화)
3

주요 개선 사항

Voice Memo 피드백을 기반으로 5개 핵심 영역을 개선하였습니다.

개선 3-1

캘린더 기반 날짜 선택

Before
단순 숫자 +/- 버튼으로 여행 일수만 선택
After
캘린더 UI에서 정확한 출발일/도착일 선택 → 예약 사이트에 날짜가 자동 반영됩니다.
캘린더 UI - 출발일 선택 대기
4/1~4/4 선택 완료 (3일간의 여행)
개선 3-2

Day 0 출발 준비 (실제 예약 링크)

Before
출발 준비 단계 없이 바로 일차별 일정으로 진행
After
  • 항공/숙박/교통/보험 4개 카테고리별 옵션 제공
  • 각 옵션에 실제 예약 사이트 "직접 확인" 링크 포함
  • 스카이스캐너/카약: ICN→CTS 날짜 포함 검색
  • Booking.com/에어비앤비: 지역별 구체적 검색
  • Klook: 교통패스/렌터카 상품 검색
항공 카테고리 - 대한항공/진에어/에어부산 직항편 비교 및 예약 링크
숙박 카테고리 - Booking.com/에어비앤비 연동
보험/준비물 카테고리 - 해외여행자 보험 및 준비 가이드
개선 3-3

카테고리 기반 일정 추천

Before
3개 옵션 중 택 1 방식
After
  • 카테고리 토글(관광/휴식/운동/미식/쇼핑/문화) 선택
  • AI가 선택된 카테고리에 맞춤 옵션 자동 생성
  • "원하는 옵션이 없나요?" 자연어 입력으로 추가 옵션 생성
기간 → 목적지 → 준비 → 카테고리 → 선택 → 완료 스텝 흐름
개선 3-4

LG 브랜드 PDF 보고서

Before
window.print() 브라우저 인쇄 기능
After
  • LG 브랜드 커버 페이지 포함
  • Day 0 준비 사항 + 일차별 일정 테이블
  • LG 브랜드 디자인 푸터
  • PDF 다운로드 기능
개선 3-5
